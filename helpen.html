If you're having a hard time browsing modiange.github.io, it's usually because of your internet connection, you can try using DNS.

Take windows 11 device as an example, other devices can be searched by yourself. The operation may affect the use of the device or other software, please operate with caution or seek professional help! Open System Settings -> Network Connections Open "Current Network Properties" -> "internet protocol version 4". Set the DNS address to 8.8.8.8 and restart the computer

You can also set it to another DNS address.
