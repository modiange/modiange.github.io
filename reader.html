Android NFC 读取器

使用 Web NFC API 在 Android Chrome 浏览器中读取 NFC 标签

使用说明

本页面使用 Web NFC API 在支持 NFC 的 Android 设备上读取 NFC 标签。请确保您的设备满足以下要求：

系统要求

  • Android 设备支持 NFC 功能
  • Chrome 浏览器版本 89 或更高
  • 设备已启用 NFC
  • 页面通过 HTTPS 访问（本地 localhost 除外）
  • 用户已授予 NFC 访问权限

支持的 NFC 标签类型

NFC-A (ISO/IEC 14443-3A) NFC-B (ISO/IEC 14443-3B) NFC-F (FeliCa) NFC-V (ISO/IEC 15693) NDEF 消息

使用技巧

  • 将 NFC 标签靠近设备背面（通常摄像头附近）
  • 保持标签与设备距离在 1-3 厘米内
  • 读取过程中不要移动设备或标签
  • 某些 NFC 标签可能需要更长的读取时间
  • 如果读取失败，请尝试重新放置标签

数据格式

  • 纯文本: 直接显示文本内容
  • URL: 自动转换为可点击链接
  • 智能海报: 解析标题、URL和类型
  • MIME 类型: 根据类型解析内容
  • 外部类型: 自定义应用数据

隐私与安全

  • NFC 读取需要用户明确授权
  • 数据仅在当前页面处理
  • 不会自动发送数据到服务器
  • 权限可以随时在浏览器设置中撤销
  • 确保使用 HTTPS 连接保护数据
